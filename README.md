# 🌍 Sistema de Cadastro de Territórios em C++

Este projeto foi desenvolvido como parte dos meus estudos em **Estrutura de Dados**, focado na organização e manipulação de informações compostas.

## 🚀 O que aprendi com este desafio

Este desafio introduziu o conceito de **structs** como uma ferramenta poderosa para agrupar dados relacionados. Durante o desenvolvimento, foquei em três pilares principais:

1.  **Estruturas Compostas (Structs):** Aprendi a criar um "molde" personalizado para representar um Território, agrupando nome (string), cor (string) e quantidade de tropas (int) em uma única unidade lógica. Isso torna o sistema muito mais legível e escalável.
2.  **Gerenciamento de Fluxo (Laços de Repetição):** Implementei laços `for` para automatizar a entrada e a saída de dados, permitindo o gerenciamento de múltiplos territórios de forma eficiente através de um array de structs.
3.  **Tratamento de Entrada de Dados:** Dominei o uso de funções como `fgets()` para leitura de strings com espaços e `strcspn()` para limpeza de buffers, garantindo que a interface no terminal seja limpa e livre de erros de memória.

## 🛠️ Tecnologias Utilizadas
- Linguagem **C/C++**
- Compilador **MinGW (g++)**
- Editor **VS Code**

*Este projeto demonstra como utilizar estruturas compostas para organizar informações e criar sistemas mais legíveis e escaláveis.*
